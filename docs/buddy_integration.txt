Buddy Allocator Integration Guide
================================

1. Include header in main.cpp:
   #include "buddy/buddy.h"

2. Create instance:
   BuddyAllocator buddy;
   bool useBuddy = false;

3. Extend init command:
   init memory <size>   -> existing Memory allocator
   init buddy <size>    -> buddy.init(size); useBuddy = true;

4. Route commands:
   malloc <size> -> if(useBuddy) buddy.mallocBlock(size); else mem.mallocBlock(size);
   free <id>     -> if(useBuddy) buddy.freeBlock(id); else mem.freeBlock(id);
   dump          -> if(useBuddy) buddy.dump(); else mem.dump();

5. Stats remain Memory-only (Buddy stats optional)

This keeps both allocators independent and clean.
